# 🌐 Personal Portfolio Website

A modern and responsive **Personal Portfolio Website** built using **HTML, CSS, and JavaScript** to showcase skills, projects, and professional profile.

---

## 🚀 Live Demo

https://adityakamble69.github.io/Portfolio/
---

## 📌 Overview

This portfolio website represents my personal and professional journey as a **Full Stack Developer**.
It highlights my **skills, projects, education, and experience** in a clean and interactive UI.

---

## 🎯 Features

### 👤 Profile Section

* Personal introduction
* Contact details (Email, Phone, Location)
* Social media links

### 📖 About Section

* Professional summary
* Skills & interests
* What I do (Services)

### 📄 Resume Section

* Education timeline
* Experience details
* Skills with progress bars

### 💼 Portfolio Section

* Project showcase
* Category-based filtering:

  * Web Design
  * Web Development
* Live project links

### 📩 Contact Section

* Contact form with validation
* User-friendly UI

---

## 🧠 Key Functionalities

* 📱 Fully responsive design
* 🎨 Dark theme UI
* ⚡ Smooth animations
* 🔍 Project filtering system
* 📂 Single Page Navigation (SPA-like)
* 📬 Form validation using JavaScript
